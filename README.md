# The Data Analyst Career Guide for Career Pivoters

## Overview
This project analyzes data analyst salaries and skill requirements to uncover which skills provide the highest value in the job market. Using Python-based analysis, the project explores how different skills (e.g., Python, SQL, Excel) influence salary outcomes and helps identify data-driven career strategies for aspiring analysts.

## Problem
Many aspiring data analyst career pivoters struggle to identify which skills they should develop and which industries provide the best starting opportunities. Our research bridges this gap by quantifying industry accessibility, salary growth potential, and job demand.

## Objectives
- Analyze salary distributions across roles and industries
- Identify most valuable skills for entry-level data analysts
