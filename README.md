# The Data Analyst Career Guide for Career Pivoters

## Overview
This project analyzes data analyst salaries and skill requirements to uncover which skills provide the highest value in the job market. Using Python-based analysis, the project explores how different skills (e.g., Python, SQL, Excel) influence salary outcomes and helps identify data-driven career strategies for aspiring analysts.

## Objectives
1. Analyze salary distributions across roles and industries
2. Evaluate impact of skills on salary
3. Identify most valuable skills for entry-level data analysts

## Tools and Technologies
1. Python (Pandas, NumPy)
2. Data Visualization (Matplotlib, Seaborn)
3. Google Colab

## Key Insights
1. The most in demand skills for data analysts are: SQL, Excel, Statistics, Python, Tableau, and R, collectively representing roughly 80% of all required skills)
2. IT Services industry represents the largest share of job openings, accounting for 18.83% of postings
3. Aerospace & Defense industry provides the highest average starting salary, reaching approximately $92,000 annually
4. Real Estate industry exhibits the largest seniority pay gap, salaries increasing by an average of 67.5% from junior to senior roles
